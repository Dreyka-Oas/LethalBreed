# 🧟 LethalBreed - Git Architecture

## Structure des branches

```
main                    → Documentation (README.md, CURSEFORGE.md)
fabric                 → Toutes les versions (1.21.11/)
fabric-1.21.11         → Contenu version 1.21.11 à la racine
```

## Organisation par loader

### Fabric
- `fabric` - Branche principale Fabric avec toutes les versions
- `fabric-1.21.11` - Code source version 1.21.11 (sans sous-dossier)

## Contenu des branches

| Branche | Contenu |
|---------|---------|

| `fabric` | Dossier `1.21.11/` avec le code complet |
| `fabric-1.21.11` | Code de la 1.21.11 directement à la racine |
